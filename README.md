# Reclaim Rewards

Reclaim and manage your rewards efficiently.

## Project Overview
This project is a high-fidelity rewards program landing page and application built with Next.js, TypeScript, and Tailwind CSS.

## Features
- Hero section with cinematic animation.
- Rewards overview and program steps.
- Eligibility and FAQ sections.
- Integrated registration form.
- Mascot-driven branding.

## Getting Started
To get started with the project, run:
```bash
npm run dev
```
